# SYNCRIG QA Verification Checklist

본 문서는 SYNCRIG 서비스의 주요 기능 요구사항에 대한 테스트 케이스 및 검증 상태를 기록한 체크리스트입니다.

| ID | 테스트 항목 (Feature) | 테스트 절차 (Steps) | 기대 결과 (Expected Behavior) | 결과 (Status) | 비고 (Notes) |
|---|---|---|---|---|---|
| **TC-01** | OAuth 연동 및 라이브러리 동기화 | 1. 런처 연동 버튼 클릭<br>2. 로그인 완료 후 동기화 수행 | 보유한 게임 목록 정보와 플레이타임 데이터가 DB에 정상 적재됨 | **PASS** | Mock API 연동 확인 |
| **TC-02** | 하드웨어 사양 등록 (CRUD) | 1. CPU, GPU, RAM, 해상도 선택 및 저장<br>2. 저장된 프로필 수정/삭제 | 입력값 유효성 검사 수행 후 정상 저장됨. 기본값 설정 변경 가능 | **PASS** | `HardwareProfileForm` UI 정상 동작 |
| **TC-03** | 유사도 가중치 매칭 알고리즘 | 1. 특정 GPU(예: 3060) 선택 후 추천 요청<br>2. 유사도 산출 결과 확인 | 가중치 계산식(GPU 50% 등)에 따라 90% 이상 일치하는 최적 프로파일 정렬 | **PASS** | 오차율 1% 미만 검증 완료 |
| **TC-04** | 대시보드 시각화 렌더링 | 1. 통합 대시보드 진입<br>2. 플레이타임 및 업적 통계 뷰어 확인 | Recharts 모사 게이지 바가 정상 렌더링되며, 색상이 SYNCRIG 테마에 매칭됨 | **PASS** | Vanilla CSS 호환 테스트 통과 |
| **TC-05** | 외부 API 장애 및 예외 핸들링 | 1. 백엔드 오프라인 유도 후 추천 클릭<br>2. 입력값 누락 후 API 요청 | 클라이언트 측 로컬 Fallback 데이터가 표출되고 에러가 무결히 처리됨 | **PASS** | `errorMiddleware` 및 `api.js` Fallback 연동 완료 |
