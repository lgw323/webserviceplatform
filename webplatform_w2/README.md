# 2주차: 웹 서비스 플랫폼 기획 및 아이디어 정의

## 1. 수행 목표
- 다면 웹 서비스 플랫폼(SYNCRIG)의 기본 비즈니스 아이디어 도출.
- Steam/Riot API 연동을 통한 파편화된 게이머 데이터 시각화의 가치 분석 및 그래픽 최적화 요구사항 수립.
- 전체적인 시스템 UI/UX 방향성 기획 및 발표 자료 작성.
