# 5주차: UI/UX 프로토타입 구현 및 사용자 흐름 정의

## 1. 수행 목표
- 4주차 기능 명세서에 정의된 핵심 화면 구조를 정적 웹 페이지(HTML/CSS/JS)로 프로토타이핑.
- 사용자 여정 지도(User Journey Map) 및 사용자 흐름(User Flow) 설계.
- 시인성 높은 대시보드 구조 및 하드웨어 등록 카드 폼 디자인 시안 마련.

## 2. 산출물 내역
- [index.html](./index.html), [style.css](./style.css), [script.js](./script.js): 하드웨어 프로필 설정 및 대시보드 통계를 시뮬레이션할 수 있는 HTML/CSS/JS 정적 프로토타입.
- [user flow.png](./user%20flow.png), [usecase.png](./usecase.png): 액터와 시스템 간의 인터랙션 및 화면 흐름도 이미지.
- [요구사항명세서_SRS.pdf](./요구사항명세서_SRS.pdf), [기능명세서_FSD.pdf](./기능명세서_FSD.pdf): 프로토타입 기능 검증의 바탕이 된 공식 설계 명세 백업.
- [2021663046_이건우.pptx](./2021663046_이건우.pptx), [2021663046_이건우.pdf](./2021663046_이건우.pdf): 5주차 과제 발표 자료.
